# 📄 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, NLP, and Machine Learning.

## 🚀 Features

- Resume PDF Upload
- Job Description Matching
- TF-IDF Vectorization
- Cosine Similarity Scoring
- Skill Extraction
- Missing Skills Detection
- ATS Feedback System

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- PyPDF2
- NLP

## 📊 ML Concepts Used

NLP Preprocessing
TF-IDF Vectorization
Cosine Similarity

## 📌 Future Improvements

Gemini/OpenAI Integration
Semantic Search
Better ATS Scoring
Resume Recommendations
